# machine_setup
My local machine setup and config 

## Homebrew installation
`/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"`


## Brew installed applications
`brew install kubectl`\
`brew install awscli`\
`brew install kops`\
`brew install nginx`\
`brew install wget`\
`brew install yarn`\
`brew install node`\
`brew install imagemagick@6 && brew link imagemagick@6 --force`\
`brew install zsh`\
`brew cask install iterm2`

## Git and github

Instead of installing git manually using git command line, download git desktop from 
[github desktop](https://desktop.github.com/) and login using github credentials which will automatically setup git locally.

## Postgres
Instead of manually installing postgres using brew or any other method download [postgres.app](https://postgresapp.com/) it will setup everything of postgres. 

## Postico
Use [postico](https://eggerapps.at/postico/) graphical client for postgres
